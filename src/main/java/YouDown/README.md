## 🚀 Como instalar e usar

**1.** Baixe o `YouDown.exe` na seção [Releases](../../releases/latest)

**2.** Clique com o botão direito → **Executar como Administrador**

**3.** Aceite os Termos de Uso na primeira execução

**4.** Na primeira abertura, o FFmpeg é detectado e instalado automaticamente

**5.** Cole a URL do YouTube, escolha o formato e clique em **Baixar Agora**

> ⚠️ **Execute sempre como Administrador** para garantir que o FFmpeg seja instalado corretamente e que o app funcione sem erros em pastas protegidas.

## 🎬 Formatos suportados

| Formato | Qualidade |
|---|---|
| **MP4** | Melhor vídeo disponível (H.264 + AAC) |
| **Melhor qualidade** | Maior resolução disponível (MKV) |
| **MP3** | Áudio extraído em qualidade máxima |
| **M4A** | Áudio nativo do YouTube |
| **WAV** | Áudio sem compressão |
